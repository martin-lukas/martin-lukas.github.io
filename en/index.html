LangReader

  • Web: www.langreader.net
  • Source: GitHub

  • A web application for studying languages through reading. Add texts you're interested in, and mark words as known, or that you want to study them. When you mark a word as studied, a translation appears above it.

CsWiktionary2RDF - extractor

  • Source: GitHub

  • An information extractor for the Czech Wiktionary, which transforms it into structured RDF data. It outputs an RDF graph, which allows querying with SPARQL. Possible uses are: word lemmatization (see the project below), enriching conjugated word forms with semantics from DBpedia (a structured Wikipedia) or finding missing information on Wiktionary.

CsWiktionary2RDF - search engine

  • Web: GH Pages
  • Source: GitHub

  • A search engine for the RDF graph from the extractor above. It searches all the Wiktionary pages for the given term, and returns all the occurences, including in which position it found on the page. For example, if we input the word obrazem, we get obraz (ɔbras): podstatné jméno, rod mužský neživotný, č. jednotné, 7. pád, which means - noun, masculine gender, singular, 7th case (instrumental). That's an example of lemmatization, where it gave us the base form of the given term, and also details about the specific form. The website is currently only available in Czech.

Föli App - bus tracking in Turku, Finland

  • Web: GH Pages
  • Source: GitHub

  • This website allows you to type in a bus number from the city of Turku, Finland. It will then show the bus route through the city as well as the current location of all the buses on that route. The location is automatically refreshed every few seconds. This project was developed for a web development course at the University of Turku.
