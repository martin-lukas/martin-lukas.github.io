LangReader

CsWiktionary2RDF - Enhanced Search Engine and Extractor on top of the Czech Wiktionary

  • An information extractor for the Czech Wiktionary - a Czech community-driven dictionary. This extractor transforms its data into a structured RDF graph, which allows querying with the SPARQL language. Possible uses are: word lemmatization (see the project below), enriching conjugated word forms with semantics from DBpedia (a structured Wikipedia) or finding missing information on Wiktionary.

    Also included is a search engine for the RDF graph from the extractor above. It searches all the Wiktionary pages for the given term, and returns all the occurences, including in which position it was found on the page. For example, if we type in the word obrazem (en. (with) an image), we get obraz (ɔbras): podstatné jméno, rod mužský neživotný, č. jednotné, 7. pád, which means - noun, masculine gender, singular, 7th case (instrumental). That's an example of lemmatization, where it gave us the base form of the given term, and also some more details about the specific form.

    The search engine website is available in English and Czech.
